These are the data validation rules I want your program to enforce:
- The cell plan must be either "Unlimited Elite", "Unlimited Extra", "Unlimited Starter" or "Prepaid"
- The autopay question must be answered either "Yes" or "No"
- The paperless billing question must be answered either "Yes" or "No"
- The number of lines on this plan must be either 1, 2, or 3
- The gigabytes question must be a value between 0 and 30 inclusive

Your program must gather the type of plan, the autopay answer, the paperless billing answer, the number of lines and the number of gigabytes, in that specific order.  However, if you detect an error in an item, you do not have to request all the remaining items if you don't want to;  just be sure you write to cout the required single error message and nothing more after that.  If instead you choose to gather all the input first before checking for any errors, and you detect more than one error, then write only the error message for the first erroneous input item as the last example shows.
